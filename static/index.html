How to use

The script

Include the script in your html code.

<script type="text/javascript" src="//texloader.herokuapp.com/tex.js"></script>

Place equations in <eq> tags

In actuality, you can place the equations in any kind of tags. If you want to use something else, you must provide the appropriate selector to the javascript function below. 

<p>Embedding Tex equations in text is easy! <eq>e^{i\pix}=cos(x)+isin(x)</eq> See?<p>

Call the convertEquations javascript function

The function has two optional parameters.

  • opt_query: The CSS selector query used to find all equation tags. If left undefined or null, it will default to "eq".
  • opt_container: The container to search in. If left undefined or null, it will default to the entire document. 
haodev.tex.convertEquations();

Selecting different equations or from a specific container will look like this: 

haodev.tex.convertEquations('span.equation', document.getElementById('container'));

The above code will turn only the text found inside span elements that have the "equation" class inside an element with id "container" into equations. Note: selected tags will be removed once the equation has been loaded.

Font Size

You may have noticed that font size isn't a required input. This is because the TexLoader script automatically uses the calculated font size for the element to determine what size the equation will be. The calculated font size is obtained at the time the javascript function is called.

Any font size is supported:

10pt font: e^{i\pix}=cos(x)+isin(x)

20pt font: e^{i\pix}=cos(x)+isin(x)

30pt font: e^{i\pix}=cos(x)+isin(x)

40pt font: e^{i\pix}=cos(x)+isin(x)

Equations are transparent PNG images

Example of a colored background cos(x)

You can directly generate an image tag using the javascript function equation

The function has two parameters.

  • equation: The tex equation to render
  • size: The font size to render it

The function returns an image object that once loaded, will contain the equation. 

var img = haodev.tex.equation('e^{x}', 24);

Images are obtained from a REST API

All images are generated by a REST API. You can directly call it if you want to.

http://texloader.herokuapp.com/tex?formula={eq}&size={sz}
  • eq: The urlencoded tex equation
  • sz: The font size of the tex equation image
